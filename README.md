# rojet-offres-emploi
Automatiser la veille des offres d’emploi IT avec un **pipeline CI/CD complet** piloté par **Jenkins** qui:  

1. extrait les offres et les stocke au format CSV
2. transforme ce CSV en une page HTML lisible
3. publie automatiquement cette page
4. se déclenche et s’arrête **automatiquement** selon la présence (ou non) de nouvelles offres.
