# rojet-offres-emploi
Automatiser la veille des offres d’emploi IT avec un **pipeline CI/CD complet** piloté par **Jenkins** qui:  1. extrait les offres et les stocke au format CSV 3. transforme ce CSV en une page HTML lisible 4. publie automatiquement cette page 5. se déclenche et s’arrête **automatiquement** selon la présence (ou non) de nouvelles offres.
